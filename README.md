# 🚀 GitHub User Dashboard

A modern and responsive React-based dashboard application that allows users to search and explore GitHub profiles using the GitHub Public API. This project demonstrates clean UI design, efficient state management, and seamless user experience with features like pagination, loading states, and error handling.

---

## 📌 Overview

The **GitHub User Dashboard** enables users to search for GitHub profiles by username and view relevant details in an organized and user-friendly interface. It is designed with performance and simplicity in mind, making it a great example of working with APIs in React.

---

## ✨ Features

* 🔍 Search GitHub users by username
* 👤 Display user profile results
* ⏳ Loading indicator while fetching data
* ⚠️ Error handling for invalid or empty searches
* 📄 Pagination (Next / Previous navigation)
* 📱 Fully responsive and modern UI

---

## 🛠 Tech Stack

* **React** (Functional Components & Hooks)
* **JavaScript (ES6)**
* **CSS**
* **GitHub Public API**

---

## 📂 Project Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/himani-stack/Github-User-Dashboard.git
```

### 2️⃣ Navigate to the project directory

```bash
cd Github-User-Dashboard
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the development server

```bash
npm start
```

---

## 🌐 Usage

1. Open the app in your browser.
2. Enter a GitHub username in the search bar.
3. Browse through the results using pagination controls.
